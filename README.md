# **telemetry-sdk**

## **Overview**
telemetry-sdk from huawei devices

## **Installation**
### **Prerequisites**

- Python : 3.10
   
### **needed modules**
- fastapi
- uvicorn
- pydantic
- grpcio
- protobuf 3.20.0

## **start up**

1. run the main.py in root dir:
   ```
   python3 main.py 
   ```
2. visit docs below by browser:   
   ```
   http://127.0.0.1:8000/docs
   ```
